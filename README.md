A fast, lightweight Moodle LMS security scanner written in Go.

### From source

Requires Go 1.21+.

```bash
git clone https://github.com/d6fault/moodlescan.git
cd moodlescan
go build -o moodlescan
```

## Usage

```bash
$ ./moodlescan scan --url https://example.com
```
