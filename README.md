A fast, lightweight Moodle LMS security scanner written in Go.

### From source

Requires Go 1.21+.

```bash
git clone https://github.com/d6fault/Evil-Moodle.git
cd moodlescan
go build -o moodlescan
```

## Usage

```bash
$ ./evilmoodle scan --url https://example.com
```
